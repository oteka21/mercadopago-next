Setup Guide

Installation

Install the package using your preferred package manager:

npm install mercadopago-next
# or
pnpm add mercadopago-next
# or
yarn add mercadopago-next

Environment Variables

Add your Mercado Pago credentials to your .env.local file:

MP_ACCESS_TOKEN=your_access_token_here
NEXT_PUBLIC_MP_PUBLIC_KEY=your_public_key_here
NEXT_PUBLIC_APP_URL=http://localhost:3000

⚠️ Get your credentials from the Mercado Pago Developers Portal

Server-Side Setup

Create a server instance in lib/mp.ts:

import { createMercadoPago } from "mercadopago-next/server";

export const mp = createMercadoPago({
  accessToken: process.env.MP_ACCESS_TOKEN!,
  publicKey: process.env.NEXT_PUBLIC_MP_PUBLIC_KEY,
  baseUrl: process.env.NEXT_PUBLIC_APP_URL,
  
  // Pre-configure products
  products: {
    basic_plan: {
      title: "Basic Plan",
      unitPrice: 999,
      description: "Basic plan description",
    },
  },
  
  // Handle webhook events
  onEvent: async (event) => {
    if (event.type === "payment.approved") {
      // Process payment
      console.log("Payment approved:", event.data);
    }
  },
});

API Route Setup

Create an API route at app/api/mp/[...all]/route.ts:

import { mp } from "@/lib/mp";

export const POST = mp.handler;
export const GET = mp.handler;

This single route handles all Mercado Pago operations including checkout, subscriptions, and webhooks.

Client-Side Setup

Create a client instance in lib/mp-client.ts:

import { createMPClient } from "mercadopago-next/client";

export const mpClient = createMPClient({
  baseUrl: "/api/mp",
});

Usage in Components

Now you can use the client in your React components:

"use client";

import { mpClient } from "@/lib/mp-client";

export default function CheckoutButton() {
  const handleCheckout = async () => {
    await mpClient.checkout({ productId: "basic_plan" });
  };

  return (
    <button onClick={handleCheckout}>
      Buy Now
    </button>
  );
}
